The key ingredients in the HEADER where everything is in a single image file:

  • The HEADER element - for semantic purposes
  • The H1 element - for semantic purposes
  • The ANCHOR element that wraps the H1 - this is a common practice; users know to click whatever's in the top-left to go back "home"
  • Content inside the H1 element: may contain words and/or an image
  • A subtitle - some text that adds more information about the website; typically, not clickable (not inside the ANCHOR element)

BTW - be careful if you need to make the logo scalable so it'll fit on mobile screens. Text might become so small it becomes unreadable!
